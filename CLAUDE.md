Portfolio templates live in `templates/`, one subdirectory per major (e.g. `templates/biology/`, `templates/electrical-engineering-b/`). Each directory contains:

- `sample.html` — the original sample website (source of truth for design)
- `mustache.html` — **autogenerated** Mustache template; do not edit directly
- `template.html` — **autogenerated** analysis template; do not edit directly
- `normalized.html` — **autogenerated** pre-processed version; do not edit directly

To change how mustache templates are generated, edit `src/netlify/functions/ExtractMustacheTemplate.md`.
To change how analysis templates are generated, edit `src/netlify/functions/ExtractExampleWebsiteTemplate.md`.

Non-template files (gallery pages, test files, marketing pages, etc.) remain in `html/`.
